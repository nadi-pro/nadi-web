# UI Components

Reusable interface elements.

## Button

Styled button component.

### Location

`src/components/Button.tsx`

### Features

- Multiple variants
- Fully typed props
- Accessible
- Supports all button attributes

## Fields

Form input field components.

### Location

`src/components/Fields.tsx`

### Features

- Text inputs
- Email inputs
- Password inputs
- Labels and validation

## Logo

Nadi logo component.

### Location

`src/components/Logo.tsx`

### Usage

```typescript
import { Logo } from '@/components/Logo'

<Logo />
```

## DarkModeToggle

Theme switcher button.

### Location

`src/components/DarkModeToggle.tsx`

### Usage

```typescript
import { DarkModeToggle } from '@/components/DarkModeToggle'

<DarkModeToggle />
```

### Features

- Toggles light/dark theme
- Visual feedback
- Keyboard accessible
- Uses next-themes
