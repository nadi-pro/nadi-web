# Documentation

## Overview

Welcome to the Nadi Web documentation. Nadi is a real-time error tracking and application monitoring platform
for developers. This web application serves as the marketing and informational website, built with modern web
technologies including Next.js 15, React 19, TypeScript, and Tailwind CSS 4.

This documentation is organized into context-based sections to help you quickly find the information you need.

## Technology Stack

- **Framework**: Next.js 15 (App Router)
- **UI Library**: React 19
- **Language**: TypeScript 5.8
- **Styling**: Tailwind CSS 4.1
- **UI Components**: Headless UI 2.2
- **Theme Management**: next-themes 0.4
